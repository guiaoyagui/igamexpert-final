# 🚀 iGameExpert – Site Institucional

💻 Aplicação Web Fullstack | React, TypeScript, APIs e Arquitetura Escalável

---

## 📌 Visão Geral

Site institucional desenvolvido para a **iGameExpert**, consultoria especializada no mercado brasileiro de iGaming.

O projeto foi criado para oferecer uma presença digital moderna, combinando **UI/UX de alta qualidade, performance e arquitetura escalável**, atendendo necessidades reais como geração de leads, apresentação de serviços e gestão de eventos.

---

## 🎯 Objetivos do Projeto

* Desenvolver um site institucional moderno e escalável
* Apresentar serviços e expertise da empresa de forma clara
* Permitir captação de leads através de formulário funcional
* Criar uma plataforma dinâmica para eventos e vagas

---

## 🧠 Problema Resolvido

Sites institucionais tradicionais costumam ser estáticos e difíceis de manter.

👉 **Solução implementada:**

* Arquitetura modular e escalável
* Conteúdo dinâmico (eventos, serviços e vagas)
* Integração entre frontend e backend
* Foco em usabilidade e experiência do usuário

---

## 🚀 Principais Funcionalidades

* 🧭 Navegação multi-página com Wouter
* 🎴 Cards interativos com efeito flip 3D nos serviços
* 📅 Sistema de eventos com filtros (próximos, passados, todos)
* 🔎 Busca de eventos por nome
* 📩 Formulário de contato funcional (geração de leads)
* 📱 Design responsivo (mobile-first)
* ✨ Animações suaves com Framer Motion

---

## 🎯 UX & Estratégia de Produto

* Estrutura de navegação clara para melhor experiência
* CTA estratégico (**Contate-nos**) presente em todas as páginas
* Hierarquia visual destacando informações importantes
* Elementos interativos para aumentar o engajamento

---

## 🛠️ Tecnologias Utilizadas

**Frontend**

* React 18
* TypeScript
* Tailwind CSS
* Framer Motion
* Wouter (roteamento)

**Backend**

* Node.js
* Express

**Comunicação**

* tRPC (comunicação tipada entre frontend e backend)

**Build Tool**

* Vite

---

## 🧩 Arquitetura

O projeto segue uma **arquitetura fullstack modular**:

* Separação entre frontend e backend
* Estrutura baseada em componentes reutilizáveis
* Comunicação via API com tRPC
* Organização escalável de pastas

---

## 📁 Estrutura do Projeto

```
client/src/
├── components/     # Componentes reutilizáveis
├── pages/          # Páginas da aplicação
├── lib/            # Utilitários e configurações
└── App.tsx         # Componente principal com rotas

server/
├── index.ts        # Servidor Express
└── routes.ts       # Rotas da API (tRPC)
```

---

## ⚙️ Como Executar o Projeto

### Requisitos

* Node.js 18+
* npm ou pnpm

### Instalação

```bash
npm install
```

ou

```bash
pnpm install
```

---

### Rodar Localmente

```bash
npm run dev
```

Aplicação disponível em:
👉 http://localhost:5000

API disponível em:
👉 http://localhost:5000/api

---

## 🏗️ Build para Produção

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta `dist/`.
